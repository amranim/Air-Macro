## 📖 What is Air-Macro?

Air-Macro is a simple but effective macro program for Windows. It records and repeats mouse and keyboard actions with high precision. The program focuses on exact timing, reliability, and low system load. It works well on Windows 10 and later versions.

Air-Macro uses low-level input methods to ensure smooth execution. It handles mouse clicks, keyboard presses, and even pixel detection. The software runs quietly in the background and keeps your PC responsive.

## 🚦 How to Use Air-Macro

After you run Air-Macro, you will see the main window. It is designed to be easy.

### Record a Macro

1. Click the "Record" button to start.  
2. Perform the actions you want to automate (mouse moves, clicks, keyboard typing).  
3. Click "Stop" when done.

The macro stores your inputs with exact timing.

### Play a Macro

1. Select the macro from the list on the main screen.  
2. Click "Play" to repeat the recorded actions.  
3. You can pause or stop playback any time.

### Adjust Timing and Settings

You can change playback speed or add delays between steps to suit your needs. The software supports precise millisecond control.

### Pixel Detection

Air-Macro can detect colors on your screen using pixel detection. This lets you create macros that wait for visual changes before continuing.

## 🔧 Troubleshooting

If Air-Macro does not respond correctly:

- Restart your Windows computer to clear input locks.
